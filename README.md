laaf
====

[![Build Status](https://secure.travis-ci.org/siciarek/laaf.png)](http://travis-ci.org/siciarek/laaf)

Light As A Feather Webservice Protocol

## Supported formats

At the moment *XML* and *JSON* are supported. *XML* is default.
To switch to JSON format you need to send HTTP header `"Content-Type: application/json"` in request.
